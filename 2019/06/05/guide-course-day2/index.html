DAY2 导游讲解技巧

讲师：王冬 东北烈士纪念馆

态势语言：

就是肢体语言。心理学家：信息有效度=7%语音语言+38%的语音语调+55%态势语言

态势语言要求：

  • 自然
  • 简洁明了
  • 适度适宜
  • 富有变化
  • 姿态端正
  • 关注游客

态势语言组成：仪表，表情语，动作语，体态

仪表：应有点正式，区别于游客的游玩性质，导游员应以职业看待。

表情语：眼睛向左看可能是回忆，向右看可能为说谎；尽量微笑，动作，口头禅重复语，习惯性动作。自己的语音和视频会体现明显，便于纠正。

动作语：

  • 头部向前：想倾听或者说服
  • 身体后倾：注意力下降

体态：靠墙站练习。

吐字归音

语音 方音方言 尾音一般明显，北京作为标准话就是因为尾音

语调

控场能力：容易被游客提问被带跑

呼气吸气：快呼慢吸 快呼快吸，慢呼快吸，慢呼慢吸。慢：20s完成，快：1s完成。

准备稿件：有声语言再创作，有感情，讲展板、讲解词中没有的东西。

稿件准备：时代背景

例如：怎么看待6 70年代的人在公园丝巾照相的现象：他们的时代导致——人共性比较强，因为文革抹杀了个性，不敢有独特的思维。这是他们这个集体的时代写照。

态度感情：敢停的为讲，而背稿是不敢停的

情景再现

对象感：眼神交流，增减词

停顿：给观众时间反应

讲解实战

导游特点：

  • 准确性

  • 逻辑性

  • 生动性

  • 口语性

  • 现场性

    有问题的争执时：问题搁置，之后处理。“你的问题很好，让我们之后详细谈”，对错争执无意义。

  • 针对性

    • 面对小学生：讲故事少道理少历史
    • 面对老一辈：讲背后的故事，少历史
    • 面对高中生：讲历史少道理少故事

    沟通4要素：观察 表达 需要 请求。

    避免指责，避免逃避责任；“我理解你现在的心情”

    人要倾听，没必要解释。

语速不要快，开头慢速抓人注意力；准确性；对象性。

方法：

  1. 制造悬念法
  2. 问答法：自问自答 我问客答 客问我答 客问客答
  3. 突出重点法
  4. 同类相似类比
  5. 妙用数字法

我的思考

讲解技能慢功夫吃经验，网课并不能提供这种。

老师很有气质，应该是年岁较大但完全看不出一丝萎靡。

人为了保持美需要牺牲很多东西，用时间做身材管理、皮肤管理、姿态管理等

不对人直接下结论，尊重其所在的时代，具有同理心。关于公园里丝巾照相的现象理解的很到位。

导游与文化传播类的讲解毕竟有所不同，但不专业的讲解（野史、八卦）一定会给人不好的感觉，应学习展馆讲解员的方式。反思自己曾作为哈军工讲解员，并没有思考这件事情，但短短两个月的讲解却带给我很多影响。
