Git上传远程分支命令组

Author: Sirice Date: April 17, 2019  12:20:57

本地代码上传到github的远程分支

$ git init
$ git add .
$ git commit -m "commit messages"

$ git branch dev2.4  // 创建本地分支
$ git checkout dev2.4 // 切换到本地分支
$ git remote add origin [address.git] // 关联远程仓库
$ git push origin dev2.4 // 上传到远程分支
